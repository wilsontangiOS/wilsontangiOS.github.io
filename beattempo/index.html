应用功能

基础功能

BeatTempo 提供精准的节拍器功能，帮助音乐家、学生和爱好者练习节奏感。

  • 精确的 BPM 控制 (20-240 BPM)
  • 多种节拍音色选择
  • 可视化节拍指示器
  • 简单直观的操作界面
高级功能

订阅高级功能，解锁更多专业训练工具：

  • 复杂节拍模式 (3/4, 6/8, 7/8 等)
  • 节奏练习模式
  • 速度渐变训练
  • 自定义节拍组合
  • 练习历史记录
适用人群

BeatTempo 适用于各种音乐学习者：

  • 乐器演奏者（钢琴、吉他、小提琴等）
  • 音乐学生和教师
  • 乐队和合唱团成员
  • DJ 和电子音乐制作人
  • 舞蹈练习者

常见问题

  • 如何订阅高级功能？
    请在应用内通过 Apple App Store 购买并激活订阅。打开应用后，点击"升级到高级版"按钮，按照提示完成购买流程。
  • 应用无法启动或闪退怎么办？
    请尝试以下解决方案：
    1. 重启您的设备
    2. 确保应用已更新至最新版本
    3. 检查设备存储空间是否充足
    4. 如果问题持续存在，请联系技术支持
  • 订阅未生效怎么办？
    请确保使用的 Apple ID 与购买时一致，并查看 RevenueCat 收据状态。如果问题仍未解决，请联系我们的支持团队并提供您的订单号。
  • 可以在多个设备上使用订阅吗？
    可以，只要您在所有设备上使用相同的 Apple ID 登录，订阅即可在所有设备上使用。
  • 如何取消订阅？
    您可以在 Apple ID 设置中管理订阅：
    1. 打开设置 > 点击您的姓名 > 媒体与购买项目 > 查看账户
    2. 找到订阅部分，选择 BeatTempo 订阅
    3. 点击"取消订阅"
